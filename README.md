Command Line Interface for Wappsto, so that it is possible to create Wapps locally.


## Install

```
$ npm install wappsto-cli --save-dev
```

## Usage

To create a Wapp:
```
$ create-wapp
```

To update the Wapp:
```
$ update-wapp
```

To delete the Wapp:
```
$ delete-wapp
```
